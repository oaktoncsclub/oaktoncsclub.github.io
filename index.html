Competitions


ACSL

We participate in the American Computer Science League!

ACSL is a yearly contest that consists of a set of 4 contests covering both computer science theory and problem solving skills. Each contest has a written portion and a programming portion, and the latter can be solved in any language you want. You can participate in the Junior, Intermediate, or Senior leagues. Sign-up forms will be posted soon!

USA Computing Olympiad

GET STARTED BY CREATING AN ACCOUNT HERE

USACO is a set of contests split into Bronze, Silver, Gold, and Platinum divisions, where you start at Bronze and automatically get promoted up as you solve problems. These problems get VERY hard, and focus on performance. You also get a strict time limit to solve them. USACO is also free to participate in, and you only need to create an account online to start.

The first contest is from December 16-19, 2017.

Visit their website here
